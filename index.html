Lesson 0: The Basics of Web and HTML - Getting Started with HTML

Intro to the Web

  • WWW - The World Wide Web is the network of networks. It provides the paths to connect to content anywhere in the world.
  • HTML - Hypertext Markup Language is the language used to define the structure and element information of a webpage. The text defines what you see and the markup defines how it looks in the browser.
  • Images, videos and music are referenced and linked to in the HTML document.
  • Browser - website viewer
  • HTTP - Hypertext Transger Protocol is the application protocol of the web. It decodes HTML into a readable form for humans.
  • Internet - The Internet is the largest computer network in the world and is comprised of all public computers and servers.
  • Servers - Servers are specialized computers that host files used for websites and other accessable contents for use on the internet.

HTML Markup and Tags

  • Opening <tag>
  • Contents - text between the tags
  • Closing Tag - </tag>
  • Element - opening, contents and closing together
  • b - bold
  • em - emphisis italics
  • a - anchor (a href="url">contents/a)
  • (img src="url" alt="text")
  • alternate text - displayes text if there is a broken request
  • br - line break
  • p - paragraph

Elements

Block elements boxify the content contained inside the block element tag. Below are some examples of both inline and block tags.

  • inline - a, br, img, span, strong
  • block - div, p - paragraph, form - height and width can be modified

How Learning Anything Works

When learning anything new you start out as a novice and apprise to one day become an expert.   In this nanodegree the instructor advises we will use a staged approach as we advance in the course and our knowlege on the subject of programming.

  1. Stage 0 - Ignorence - novice with virtually no knowledge of the subject
  2. Stage 1 - Awareness - very limited functional knowledge, will recognise basic terms - minutes to aquire
  3. Stage 2 - Ability - working knowledge of the subject, can work independently utilizing resources - days to aquire
  4. Stage 3 - Fluency - expert in field - takes years to aquire

How We Will Use the Stages

Fluency is not the goal of this course, we will shoot for awareness and ability in several areas.

  • Awareness of SQL Databases, Get vs Post Requests and Webpage Structure.
  • Ability to use Functions and Abstract Thinking

Throughout the course when feeling frustrated we will ask for help.   Some good resource for help are Google and the Udacity discussion forum; or sometimes taking a break is all you will need.

Lesson 1: Make Your Webpage - Creating a Structured Document with HTML

Structured HTML Documents

The structure of any HTML document is very important. It is the foundation for the website and will be the basis from which all others aspects of the site are developed. A good baseline stucture will avoid repetition in you HTML, CSS and Javascript code, which will make changes or expansion to the website much easier and the website will run efficiently.

Three Languages Web Developers Must Learn!!!

  1. HTML - HyperText Markup Language: The language of the internet. Defines the basic structure of web pages.
    • Browsers Uses syntax rules to interpret HTML
    • Language of the web
    • Contains content text
    • Contains links to other web pages, images, videos or documents
  2. CSS - Cascading Style Sheets: Makes declarations regarding style and layout for TAGs used to display web page content. Using Class tags alow easy chages to fonts, text, backgrounds and other styling aspects of your website.
    • Color
    • Borders
    • Font
    • Background
  3. JS - JavaScript: Implements interactive components in a web page.
    • Can change HTML Images
    • Enables users to make changes to the website
    • Used to vaildate information input to the website
